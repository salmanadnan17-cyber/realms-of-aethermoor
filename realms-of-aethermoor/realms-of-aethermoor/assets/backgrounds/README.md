# Backgrounds

Place your world splash screen and background images here.

## Recommended Files

- `world-splash.jpg` - Main world splash screen (1920x1080 recommended)
  - This image is shown when selecting the world

## Free Resources

- [Unsplash](https://unsplash.com/) - Free high-resolution photos
- [Pixabay](https://pixabay.com/) - Free images and illustrations
- [ArtStation](https://www.artstation.com/) - Fantasy art (check licenses)

## Image Specifications

- Format: JPG or PNG
- Recommended size: 1920x1080 or higher
- Keep file sizes reasonable (<2MB) for faster loading
