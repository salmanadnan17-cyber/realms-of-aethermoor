# Tokens

Place your token images here for NPCs, monsters, and PCs.

## Token Format

- Format: PNG (with transparency recommended)
- Recommended size: 400x400 pixels
- Circular or square formats work well
- Keep file sizes small (<200KB per token)

## Free Token Resources

- [Token Stamp 2](https://rolladvantage.com/tokenstamp/) - Create tokens from images
- [Forgotten Adventures](https://www.forgotten-adventures.net/token-packs) - Free token packs
- [2-Minute Tabletop](https://2minutetabletop.com/product-category/tokens/)
- [Devin Night](https://immortalnights.com/tokensite/) - Free token collection

## Organization Tips

Consider organizing tokens in subdirectories:
- `npcs/` - Non-player characters
- `monsters/` - Creatures and enemies  
- `objects/` - Interactive objects
- `pcs/` - Player character options
