## Creating Maps

### Free Tools
- [Dungeondraft](https://dungeondraft.net/) - Paid but popular
- [Inkarnate](https://inkarnate.com/) - Free tier available
- [Dungeon Scrawl](https://dungeonscrawl.com/) - Free online tool

### Free Map Assets
- [2-Minute Tabletop](https://2minutetabletop.com/)
- [Crosshead Studios](https://www.crossheadstudios.com/)
- [Forgotten Adventures](https://www.forgotten-adventures.net/)

## Map Specifications

- Format: JPG or PNG
- Grid: 140 pixels per 5ft square (standard for Foundry VTT)
- Recommended size: 3000x2000 pixels or higher
- Include grid or gridless versions
