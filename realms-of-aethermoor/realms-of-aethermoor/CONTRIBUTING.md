# Contributing to Realms of Aethermoor

Thank you for your interest in contributing to this Foundry VTT world!

## How to Contribute

### Reporting Issues
- Check existing issues before creating a new one
- Provide clear descriptions and steps to reproduce
- Include your Foundry VTT version and dnd5e system version

### Suggesting Enhancements
- Describe the enhancement in detail
- Explain why it would be useful
- Consider how it fits with the existing world lore

### Submitting Changes

1. **Fork the Repository**
   ```bash
   git clone https://github.com/yourusername/realms-of-aethermoor.git
   cd realms-of-aethermoor
   ```

2. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

3. **Make Your Changes**
   - Follow the existing file structure
   - Maintain JSON formatting
   - Keep lore consistent with the world

4. **Test Your Changes**
   - Import the world into Foundry VTT
   - Verify all JSON files are valid
   - Test new features/content

5. **Commit Your Changes**
   ```bash
   git add .
   git commit -m "Add: Brief description of changes"
   ```

6. **Push and Create Pull Request**
   ```bash
   git push origin feature/your-feature-name
   ```

## Content Guidelines

### NPCs
- Include complete stat blocks
- Provide personality traits, ideals, bonds, and flaws
- Make sure they fit the world's tone and lore

### Items
- Balance items appropriately for D&D 5e
- Tie items to the world's lore when possible
- Include clear descriptions

### Journal Entries
- Maintain consistent voice and tone
- Provide useful information for GMs
- Keep formatting clean and readable

### Scenes
- Include appropriate grid settings
- Note any special features or hazards
- Provide context for when to use the scene

## Code of Conduct

- Be respectful and constructive
- Welcome newcomers
- Focus on what's best for the community
- Show empathy towards other contributors

## Questions?

Feel free to open an issue for questions about contributing!
